🚀 Just shipped a production-ready Data Science product from scratch — and I'm incredibly proud of it.

📊 Introducing: Smart Business Analytics Dashboard with AI Insights

A full-stack ML-powered web app that transforms any CSV into actionable intelligence — in seconds.

─────────────────────────────────────────────────────────────────
🔥 WHAT IT DOES
─────────────────────────────────────────────────────────────────
Upload any CSV and get:

✅ Automated data cleaning (null handling, deduplication, outlier detection)
✅ Full EDA — correlation heatmaps, distribution plots, trend analysis, pair plots
✅ Auto-detected ML task (classification vs regression)
✅ 3 trained models — Logistic/Linear Regression, Random Forest, and XGBoost
✅ Live single-row predictions with class probabilities
✅ AI-generated, plain-English business insights exported as a report

─────────────────────────────────────────────────────────────────
🛠️ TECH STACK
─────────────────────────────────────────────────────────────────
🐍 Python · Pandas · NumPy
🤖 Scikit-learn · XGBoost
📊 Plotly · Seaborn · Matplotlib
🖥️ Streamlit (multi-page, dark-themed UI)

─────────────────────────────────────────────────────────────────
🧠 WHY I BUILT THIS
─────────────────────────────────────────────────────────────────
Most companies sit on piles of CSV data but lack a fast way to go from raw data → insight → decision.

I wanted to build a tool that any analyst or business user could hand a CSV to and get:
→ A cleaned, profiled dataset
→ A trained predictive model
→ Concrete, plain-English insights

No code required on the user's end.

─────────────────────────────────────────────────────────────────
📈 RESULTS (tested on Mental Health in Tech Survey, 1,259 rows)
─────────────────────────────────────────────────────────────────
• XGBoost achieved 72.2% classification accuracy (no hyperparameter tuning)
• Full pipeline: upload → prediction → insight in under 60 seconds
• AI Insights engine surfaces 7+ actionable findings automatically

─────────────────────────────────────────────────────────────────
📦 PROJECT ARCHITECTURE
─────────────────────────────────────────────────────────────────
├── app.py               → Streamlit UI (5 pages)
├── data_processing.py   → ETL, cleaning, encoding
├── model.py             → Training, evaluation, prediction
└── utils.py             → Visualizations + AI insights engine

Clean separation of concerns. Production-grade error handling. Ready to deploy on Streamlit Cloud in 3 clicks.
